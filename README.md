# Automated Tasbeeh Counter

A simple and elegant web application that automatically counts Dhikr (Islamic phrases) sequentially. Built with HTML, CSS, and JavaScript.

## Features
- **Auto-Counting:** Smooth transitions between phrases (Subhan Allah ➡️ Alhamdulillah ➡️ Allahu Akbar) up to the final Dhikr.
- **Dynamic Colors:** Text colors change automatically with each phase.
- **Error Prevention:** The start button disables while running to prevent overlapping counts.
- **Responsive:** Looks great and works perfectly on both desktop and mobile devices.

## How to Run
No installation or server is required. Just open the `index.html` file in any web browser and click **Start**.
